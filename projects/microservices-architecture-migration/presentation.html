AWS Services Breakdown

Compute & Orchestration

  • EKS: Managed Kubernetes clusters
  • EC2: Worker nodes for EKS
  • Fargate: Serverless containers (optional)
  • Auto Scaling: Cluster and pod autoscaling

Networking

  • VPC: Isolated network environment
  • ALB: Application Load Balancer
  • CloudFront: Global CDN
  • Route 53: DNS management
  • App Mesh: Service mesh for microservices

Databases

  • RDS PostgreSQL: Multi-AZ for transactional data
  • DocumentDB: MongoDB-compatible
  • DynamoDB: High-throughput NoSQL
  • Elasticsearch: Search and analytics
  • ElastiCache: Redis for caching

Messaging

  • MSK: Managed Kafka for event streaming
  • SQS: Message queuing
  • SNS: Event notifications

Security

  • AWS WAF: Web application firewall
  • AWS Cognito: Service-to-service auth
  • Secrets Manager: Credential storage
  • IAM: Role-based access control
  • VPN Gateway: Secure access

Monitoring & Observability

  • CloudWatch: Metrics & monitoring
  • X-Ray: Distributed tracing
  • DataDog: APM and monitoring
  • Grafana: Dashboards
